# Book Focus

## ✨ 개요 (Overview)

* 독서에 집중을 못하는 이들을 위한 앱입니다.
* Cool한 독서앱을 추구합니다.

## 🧩 주요 기능 (Features)

* 화이트 노이즈와 Focus 모드
* 읽은 책을 기록하고 책을 읽는데에 걸린 시간 기록
* 책 읽는 시간과 책 개수를 종합하여 레벨링 시스템

## 🛠 기술 스택 (Tech Stack)

* Language: Kotlin
* Framework: Android

## 📧 커밋 메시지(Commit message)

Type
* feat : 새로운 기능 추가
* fix : 버그 수정
* docs : 문서 수정
* style : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
* refactor : 코드 리펙토링
* test : 테스트 코드, 리펙토링 테스트 코드 추가
* chore : 빌드 업무 수정, 패키지 매니저 수정

type : subject
body 
footer
